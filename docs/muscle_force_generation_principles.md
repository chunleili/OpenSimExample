# 肌肉发力的生物学原理与 Hill-type Muscle 模型

## 一、肌肉的微观结构

肌肉的层级组织：

```
肌肉 (Muscle)
 └── 肌束 (Fascicle)
      └── 肌纤维 (Muscle Fiber) ← 一个细胞
           └── 肌原纤维 (Myofibril)
                └── 肌节 (Sarcomere) ← 最小功能单位
```

肌节是肌肉产力的最小功能单位，由两种蛋白丝组成：
- **粗丝**：肌球蛋白 (myosin)，上面伸出横桥 (cross-bridge)
- **细丝**：肌动蛋白 (actin)，上面有横桥结合位点

## 二、滑动丝理论（Sliding Filament Theory）

肌肉收缩并非蛋白丝本身缩短，而是粗丝和细丝**相对滑动**，使肌节长度缩短：

```
松弛态：
   Z盘 ─────── 细丝 ═══════ 粗丝 ═══════ 细丝 ─────── Z盘
                  ←── 肌节长度 ──→

收缩态：
   Z盘 ──── 细丝 ══ 粗丝 ══ 细丝 ──── Z盘
              ←─ 缩短 ─→
```

## 三、横桥循环（Cross-bridge Cycle）

横桥循环是肌肉产力的分子引擎，一个循环包含四步：

```
1. 结合：横桥与肌动蛋白结合（需要 Ca²⁺ 暴露结合位点）
2. 摆动（Power Stroke）：横桥头部旋转，拉拽细丝滑动，释放 ADP + Pi
3. 脱开：新的 ATP 结合，横桥与肌动蛋白分离
4. 复位：ATP 水解为 ADP + Pi，横桥回到高能构象，准备下一次结合
```

肌肉总力 = 同时处于"结合并产力"状态的横桥数量 × 每个横桥的力

### 神经激活到产力的信号通路

```
运动神经元放电
  → 神经肌肉接头释放乙酰胆碱
  → 肌纤维膜去极化
  → T小管传导电信号至肌浆网
  → 肌浆网释放 Ca²⁺
  → Ca²⁺ 与肌钙蛋白 (troponin) 结合
  → 原肌球蛋白 (tropomyosin) 构象改变，暴露肌动蛋白结合位点
  → 横桥循环启动 → 产力
```

这个过程存在时间延迟，对应 Hill 模型中的 **activation dynamics**。

## 四、肌肉产力的三大关系

### 4.1 Force-Length 关系

肌肉能产生的力取决于当前纤维长度：

```
主动力
  ↑        ╱╲
  │       ╱  ╲
  │      ╱    ╲
F₀│─────╱──────╲─────
  │    ╱   最优  ╲
  │   ╱   纤维长度 ╲
  └──╱──────────────╲──→ 纤维长度
                     + 被动力（PE）在长纤维区叠加
```

**生物学解释**：
- **最优长度**：粗丝和细丝重叠面积最大，可形成横桥数量最多
- **过短**：细丝相互碰撞重叠，干扰横桥结合
- **过长**：粗丝细丝重叠区减小，可用横桥结合位点变少

### 4.2 Force-Velocity 关系

肌肉能产生的力取决于收缩速度：

```
力
↑
1.5F₀ ┐ (离心区)
      │╲
  F₀ ─┤──╲──────── ← 等长 (v=0)
      │    ╲
      │      ╲
      │        ╲
      └──────────╲─→ 缩短速度
      0          V_max
```

**生物学解释**：
- **等长 (v=0)**：横桥有充足时间结合，结合数最多，主动力最大
- **缩短 (v>0)**：结合位点快速滑过，横桥来不及结合，速度越快力越小
- **离心 (v<0)**：已结合横桥被强行拉开，产生超过 F₀ 的被动抵抗力
- **V_max**：横桥完全来不及结合，主动力降为零

### 4.3 Activation Dynamics

神经激励 (excitation, u) 到肌肉激活 (activation, a) 之间存在一阶动力学延迟：

```
da/dt = f(u, a)
```

- 激活（Ca²⁺ 释放）较快，时间常数 ~10 ms
- 去激活（Ca²⁺ 回收）较慢，时间常数 ~40 ms
- 这意味着肌肉不能瞬时开关，存在"响应延迟"

## 五、Hill-type Muscle 模型

### 5.1 模型结构

Hill 模型用三个力学元件近似肌肉的力学行为：

```
            ┌─── CE（收缩元件）───┐
            │                     ├─── SE（串联弹性元件）──→ 骨骼
            └─── PE（并联弹性元件）─┘
```

| 元件 | 物理对应 | 功能 |
|------|---------|------|
| CE (Contractile Element) | 肌纤维中横桥的主动拉拽 | 主动产力的"马达"，受 f_L、f_V、activation 调节 |
| PE (Parallel Elastic Element) | 结缔组织、肌膜、titin 蛋白 | 被动弹性，纤维被拉长超过最优长度时产生回弹力 |
| SE (Series Elastic Element) | 肌腱 | 串联弹簧，传递力并存储/释放弹性势能 |

### 5.2 产力公式

肌肉沿肌腱方向的总力：

```
F_tendon = F_muscle × cos(α)

F_muscle = F_CE + F_PE + F_damping

其中：
  F_CE      = F₀ × a × f_L(l̃) × f_V(ṽ)     主动力
  F_PE      = F₀ × f_PE(l̃)                   被动力
  F_damping = F₀ × d × ṽ                      阻尼力

  F₀  = max_isometric_force
  a   = 激活度 (0~1)
  l̃   = 归一化纤维长度 = l_fiber / l_optimal
  ṽ   = 归一化纤维速度 = v_fiber / (V_max × l_optimal)
  α   = 羽状角 (pennation angle)
  d   = fiber_damping 系数
```

### 5.3 肌腱力学

肌腱被建模为非线性弹簧：

```
F_tendon = f(ε)

ε = (l_tendon - l_slack) / l_slack    （肌腱应变）
```

- ε < 0：肌腱松弛，不产力
- ε > 0：肌腱拉伸，力随应变非线性增大
- `tendon_strain_at_one_norm_force` 定义了 F = F₀ 时的应变（即刚度参数）

刚性肌腱 (`ignore_tendon_compliance = True`) 假设肌腱不可伸缩，简化计算但丢失弹性储能效应。

### 5.4 羽状角（Pennation Angle）

肌纤维不一定与肌腱方向平行：

```
        肌腱方向
        ←─────────────────
       ╱ α (羽状角)
      ╱
     ╱  肌纤维方向
```

- 有效力 = 肌纤维力 × cos(α)
- 角度越大，单根纤维沿肌腱方向的有效力越小
- 但更大的角度允许更多纤维并排排列，增加总横截面积
- 羽状角随纤维长度变化（纤维缩短时角度增大）

### 5.5 OpenSim DeGrooteFregly2016Muscle 参数总结

| 参数 | 含义 | 对应元件 |
|------|------|---------|
| `max_isometric_force` (F₀) | 最大等长收缩力 (N) | CE |
| `optimal_fiber_length` (l_opt) | 产生最大主动力的纤维长度 (m) | CE, PE |
| `max_contraction_velocity` (V_max) | 最大缩短速度 (l_opt/s) | CE |
| `pennation_angle_at_optimal` (α₀) | 最优长度时的羽状角 (rad) | CE |
| `fiber_damping` (d) | 纤维粘性阻尼系数 | CE |
| `tendon_slack_length` (l_slack) | 肌腱松弛长度 (m) | SE |
| `tendon_strain_at_one_norm_force` (ε₀) | F₀ 时肌腱应变 | SE |
| `ignore_activation_dynamics` | 是否忽略激活延迟 | 激活动力学 |
| `ignore_tendon_compliance` | 是否忽略肌腱弹性 | SE |
| `tendon_compliance_dynamics_mode` | 肌腱动力学求解方式 | SE |

## 六、真实运动中超越等长极限的机制

Hill 模型描述的是单块肌肉单一瞬间的力学行为，真实运动可通过以下机制产生更大的输出：

1. **拉长-缩短循环 (SSC)**：先离心后向心，离心阶段的超额力 + 弹性势能释放叠加
2. **肌腱弹弓效应**：肌腱慢慢积累能量、瞬间释放，峰值功率远超肌肉本身
3. **神经适应**：训练提高运动单元募集率和放电同步性，使激活度更接近 1.0
4. **多关节动力链**：多块肌肉依次发力，力量沿运动链层层叠加传递
